## Instalar Frontend

### `npm install`

## Modificar APIURL

se encuentra en la ruta
 `/src/url.js`
si se ejecuta en local cambiar por
`export const APIURL = 'http://localhost:8080';`

o dejar la ruta de la API desplegada.

`export const APIURL = 'http://54.85.253.46:8080':`

## Ejecutar Frontend

### `npm run`
